MASHUPS VIAJECITOS: Sistema de Gestión de Vuelos con SOAP en Java

  • Herramientas: Java, SOAP (JAX-WS), Google Maps Platform, Maven, REST APIs, JDK 11+, GitHub
  • Repositorio: Ver en GitHub

Este proyecto desarrolla un sistema de gestión de vuelos que integra Google Maps Platform mediante un servicio intermedio SOAP en Java. Permite calcular rutas, distancias, tiempos de viaje ajustados por huso horario y visualizar vuelos con escalas en mapas web o móviles.

Se implementa un servicio SOAP que encapsula las APIs REST de Google Maps (Directions, Time Zone, Geocoding), protegiendo la clave de API y facilitando la integración con clientes que requieren SOAP. La lógica considera tramos de vuelo, tiempos de conexión y muestra itinerarios completos con horarios locales y distancias por tramo.

Este proyecto representa una solución escalable y segura para la gestión de vuelos, ofreciendo un backend robusto en Java y SOAP que permite integraciones con diferentes tipos de clientes mientras mantiene control sobre la API Key y la lógica de cálculo de rutas.
